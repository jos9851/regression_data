Regression Analysis on Conventional and Social Media 2014 and 2015 dataset

Abstract

Using regression analysis on the “CSM (Conventional and Social Media Movies) 2014 and 2015 dataset”, this study delves into the complex relationship between conventional and social media factors in predicting a movie’s success. By examining variables such as Genre, Budget, Number of Screens, Sequel status, Rating, and Aggregate Actor Followers, the research aims to equip producers with the information needed to strategically allocate resources. The insights gained from this analysis can inform decisions on budgeting for social media advertising, optimizing marketing efforts, and ultimately contribute to the overall economic success of movies. By bridging the gap between data-driven analysis and practical applications, this study provides valuable tools for decision-making amidst the constantly evolving landscape of the film industry.

Introduction

  • I obtained this dataset, “CSM (Conventional and Social Media Movies) 2014 and 2015 dataset” from UC Irvine Machine Learning Repository. Here’s a link to the dataset: ‘http://archive.ics.uci.edu/static/public/424/csm+conventional+and+social+media+movies+dataset+2014+and+2015.zip’

  • The original dataset was obtained by Mehreen Ahmed from the National University of Sciences and Technology, Islamabad, Pakistan (Ahmed et al., 2015) from Youtube, Twitter, and IMDB.

  • The data mostly represents popular movies mostly Hollywood movies.

  • The original data set was obtained to prove that social media features can predict the success of movies and even with a blend of conventional features we can predict more accurately the success of movies.

  • The data was collected from various sources, including movie websites like IMDB, general web resources like Wikipedia, and social media platforms such as YouTube and Twitter. To capture dynamic data like follower counts on Twitter and ratings on IMDB, the project uses sentiment analysis libraries and regularly collects the latest data using APIs and scrapers, bypassing the use of pre-existing movie datasets. (Ahmed et al., 2015)

Data Characteristics

The Data set comprises 231 movies with 15 variables. The variables are further described below.

Variables

Response

  • Gross: Total amount of money a movie makes before deductions. This is a numerical variable.

Predictors

  1. Genre: 19 categories of Genre were considered such as Action, Adventure and Drama etc. Each nominal value of genre was mapped onto numeric value ranging from 1-19 in order to improve the performance of the learning algorithms. This is a categorical variable.

  2. Budget: This numerical variable represents the production cost of movies.

  3. Number of Screens: Total count of cinema screens showing a movie. It indicates how widely shown and popular a movie is. This is also a numerical variable.

  4. Sequel: The value represents whether the movie is sequel or an individual. It is a numeric number that ranges from 1 to N. 1 shows that movie is first release (no sequel), whereas 2 represents that the movie is 2nd in a sequel; E.g. Pirates of Caribbean: Dead Man’s Chest is 2nd in sequel, therefore it is assigned the value of 2.This is a numerical variable.

  5. Rating: The value of Ratings ranges between 1 to 10 with 1 being lowest and 10 the highest. These values are collected from IMDB. This is numerical variable.

  6. Aggregate Actor Followers: Number of followers of top 3 cast. This is numerical variable.

  7. Number of views: Number of views of the Trailer on youtube.This is numerical variable.

  8. Number of likes: Number of likes of the Trailer on youtube.This is numerical variable.

  9. Number of dislikes: Number of Dislikes of the Trailer on youtube.This is numerical variable.

  10. Number of comments: Number of comments of the Trailer on youtube.This is numerical variable.

  11. Sentiment Score: This feature is represented by signed integer value. 0 represents neutral sentiment, “+”sign shows the positive sentiment whereas number shows the magnitude. Similarly “–”sign shows negative sentiment. This is numerical variable.

  12. Year: Movie Production Year. This is a categorical variable.

We then factor the Categorical variables.

Prior Theories

  1. Positive relationship between Numbers of Screen and Gross; The more cinema screens showing the movie the more income the movie rakes in.

  2. Positive relationship between Budget and Gross, If a lot of money is spent producing a movie, it is likely going to blow up and many people will pay to see.

  3. Genre is not statistically significant. The genre of a movie should not be important in determining how much a movie makes. Top movies are not from a particular genre, they could be romantic or adventurous.

  4. Aggregate actor followers should not be significant in predicting Gross. So many movies have made a lot of income while casting actors who were not so popular on social media.

  5. Dislikes should have a negative relationship with Gross. That is if many people dislike a movie then should it not make a lot of income, rather it should have a negative effect on Gross income.

SUMMARY

This project proves that through multi linear regression, we can predict the Gross income of a movie by using Ratings, Budget, Screens, Sequel, Dislikes and Aggregate Followers. One of the key takeaways is that though one would think that dislikes would affect our model negatively. That is the higher the dislikes the lesser the income the movie would make. But we can see that the reverse is the case according to our model. This also makes sense because if a movie as a lot of dislikes then it is likely that a lot of people have actually paid to see the movie. So the movie still rakes in a lot of money despite the fact that it was Disliked a lot on social media.

CONCLUSION

Looking back at our prior theories, there is a Positive relationship between Numbers of Screen and Gross; proving that the more cinema screens showing the movie the more income the movie rakes in.

Also, Positive relationship between Budget and Gross, If a lot of money is spent producing a movie, it is likely going to blow up and many people will pay to see and this is also evident in our model.

Genre is not statistically significant. The genre of a movie should not be important in determining how much a movie makes. Top movies aren’t from a particular genre, they could be romantic or adventurous. This is also true.

However, we predicted that Aggregate actor followers should not be significant in predicting Gross but our model disproves this. Which also makes sense because if a movie has a lot of popular actors then a lot of people will likely see the movie because they love the actors and follow them on social media.

Also, one of our prior theories also was dislikes should have a negative relationship with Gross. This was also disproved by our model and this was explained in the summary.
